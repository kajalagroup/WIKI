    Debugging Django emails

    November 2, 2021
    Development, Django
    Django, backend

    Run in terminal:

        python -m smtpd -n -c DebuggingServer localhost:1025

    After that in Django’s settings.py:

        EMAIL_HOST = 'localhost'
    EMAIL_HOST_PASSWORD = ''
    EMAIL_HOST_USER = ''
    EMAIL_PORT = 1025
    EMAIL_USE_SSL = False
